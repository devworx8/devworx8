# SOA Website Images

This folder should contain the following images:

## Required Images

### 1. `soa-logo.png`
The official Soil of Africa logo - Africa continent shape (green) with tree roots extending below.
- Recommended size: 200x200px or larger
- Format: PNG with transparent background

### 2. `president.jpg`
Photo of the SOA President for the ID card preview.
- Recommended size: 200x250px or larger
- Format: JPG or PNG

## Usage

These images are used throughout the website:
- `soa-logo.png`: Header, Footer, Landing page hero, ID Card
- `president.jpg`: ID Card preview page

## Placeholder

Until real images are added, the website will show broken image placeholders.
To add images, simply place them in this folder with the exact filenames listed above.
